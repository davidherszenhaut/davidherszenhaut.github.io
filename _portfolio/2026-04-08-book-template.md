**book-template** is a tool that can be used to convert a collection of Markdown files (such as chapters of a book) and a cover image into a formatted EPUB or PDF book. The template repository includes GitHub Actions workflows to create books on the web, in addition to the Makefile targets that can be used locally.

This project was built using [Pandoc](https://pandoc.org/) and [GNU Make](https://www.gnu.org/software/make/).
